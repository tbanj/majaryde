# majaryde

This is a MVP app for a ride hailing app for customers to order a ride. A user can book a ride, chat with the driver, update his details.

This project have 9 major Layouts

1. Home Layout
2. BookRide Layout
3. ComfirmRide Layout
4. SelectDriver Layout
5. TrackRide Layout
6. Chat Layout
7. Profile Layout
8. RideHistory Layout
9. CustomerOnboarding Layout
